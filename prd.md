# Цель

Создать сервис, который будет периодически (раз в час) проверять мой плейлист на YouTube, проверять наличие новых видео. Если есть новые видео, то получает субтитры видео и дает общий обзор видео, а также выделяет его главную идею. Присылает этот отчет в Telegram через бота, которого я создам.

# Стек
- локальный n8n
- Docker compose
- локальная LLM через Ollama
- сервис supadata.ai для получения субтитров видео

# Требования
- Локальная модель должна загружаться, если были получены субтитры видео.
- Локальная модель должна выгружаться из памяти после того, как отчет отправлен в Telegram.
- За один раз может быть обработано много видео.
- В телеграм сообщении должна быть ссылка на видео, общий обзор видео и выделенная главная идея.
